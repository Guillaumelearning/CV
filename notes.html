Notes personnels pour HTML

Balise meta

Elle permet d'intégrer des données supplémentaires au document HTML

L'attribut charset permet de spécifier l'encodage de la page. Le plus utilisé est l'UTF-8 car il englobe tous les langages internationaux.

On pourra ajouter name=keywords/description/author avec content="*blank*" afin de donner des indications aux moteurs de recherche concernant le contenu du site internet et aussi les mots clés pour aider au référencement

Les différents tags

Mettre en italique en utilisant surtout <em> plutôt que <I> parce que ce dernier ne met que en forme alors que <em> donne plus d'information
Cela vaut aussi pour <b> et <strong> où il faudra surtout utliser <strong> pour mettre en gras

Introduire des liens

Utiliser le tag "a" avec href le plus souvent. Il ne faut surtout pas oublier d'ajouter en les balises "a" du texte ou une image pour avoir la cible du lien

Notes personnels pour CSS

Général

Le code CSS peut être intégré directement dans la balsie en question. Utile quand il faut juste changer un élément.
Dans le cas où il faut changer plusieurs éléments il est plus judicieux de créer un code CSS à part au niveau de head en le spécifiant et en ajoutant {} dans lesquels on ajoutera le code en question pour changer le style.
Ne pas hésiter à se référer à devdoc.io pour la documentation

Dans le CSS, la hiérachie d'application du code CSS commence par le "inline" (code mis directement dans la balise) puis "internal" (code mis dans head et qui renvoie à des balises) et enfin "external" qui correspond à un fichier externe qui regroupe tout le code CSS appliqué à la page et dont on fait appel avec "link"

Syntaxe CSS = selector { property : value; }

Le code CSS prend en priorité les tags avec ID ou class par rapport à un tag neutre s'il y a deux code CSS qui renvoie pour la même balise

On ne peut avoir qu'une occurence d'un "ID" alors que pour "class" on pourra le placer à plusieurs endroits. On utulisera "ID" pour les éléments qui apparaitront qu'une seule fois et "class" pour les éléments multiples et qu'on veut regrouper pour les modifier tous ensemble

On peut ajouter plusieurs "class" à une balise en les mettant l'un après l'autre avec un espace entre. Cela n'est pas possible pour "ID" où la balise ne peut pas avoir qu'un seul "ID".
